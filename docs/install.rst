Installing Aetas
================

The easiest way to install |Aetas| is with PyPi.

.. code-block:: bash

    pip install aetas

To get the very latest version of the software, install it directly from GitHub.
    
.. code-block:: bash

    pip install git+https://github.com/JoshuaPovick/aetas


Dependencies
============

- numpy
- scipy
- astropy
- matplotlib
- dlnpyutils
